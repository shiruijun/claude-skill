# claude-skill

Personal Claude Code skills and configurations.

## 一键安装

运行以下命令安装所有 skills 到你的 Claude Code 项目：

```bash
curl -sSL https://raw.githubusercontent.com/shiruijun/claude-skill/main/install.sh | bash
```

或者手动安装单个 skill：

```bash
# 下载 skill 文件到 ~/.claude/skills/ 目录
curl -sSL https://raw.githubusercontent.com/shiruijun/claude-skill/main/skills/<skill-name>.md -o ~/.claude/skills/<skill-name>.md
```
