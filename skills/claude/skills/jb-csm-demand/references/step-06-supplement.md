# Step 6: 收集补充信息

## 策略

用于 RICE 评分和优先级判断。

## 提问清单

| 序号 | 问题 | 选项 |
|------|------|------|
| 6.1 | 这个需求是否**阻断性**的？ | 是（客户无法继续操作） / 否 |
| 6.2 | **紧急程度**？ | 紧急（本周需响应）/ 高（本月内）/ 中（下季度）/ 低 |
| 6.3 | 预计影响**多少客户**？ | 仅1家 / 2-5家 / 6-20家 / 20+家 |
| 6.4 | 是否有**竞品已支持**该功能？ | 是（请列出品名）/ 否 / 不清楚 |
| 6.5 | 客户是否明确表示**影响续费/增购**？ | 是 / 否 |
