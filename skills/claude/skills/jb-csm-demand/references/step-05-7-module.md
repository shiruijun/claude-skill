# Step 5.7: 功能模块猜测（AI 自动）

## 策略

基于场景描述中的关键词，AI 自动从 Base 功能模块选项中猜测最匹配的一级/二级模块，展示给 CSM 确认。

## 猜测规则

从场景描述中提取关键词匹配：

| 关键词 | 一级功能模块 | 二级功能模块 |
|--------|-------------|-------------|
| 报销、费用申请、费用标准 | 费控与预算 | 费用申请 / 费用报销 / 费用标准 |
| 预算、预算编制、预算中心 | 费控与预算 | 预算中心 |
| 审批、我审批的、单据 | 费控与预算 | 单据管理 / 我审批的 |
| 付款、收款、借支、退款 | 财务中心 | 付款管理 / 收款管理 / 借支管理 / 退款申请 |
| 记账、会计、凭证、核算 | 财务中心 | 记账流水 / 智慧会计 / 经营中心核算 |
| 资金、银行、现金 | 财务中心 | 资金管理 |
| 存货、资产、摊销、工资 | 财务中心 | 存货管理 / 资产管理 / 摊销与计提 / 工资管理 |
| 报表、利润表、预实、资金变动表 | 数据中心 | 常规贡献式利润表 / 预实汇总表 / 资金变动表 / 驾驶舱 |
| 供应商、客户、产品线、项目、经营单元 | 基础设置 | 基础档案-供应商 / 基础档案-客户 / 基础档案-产品线 / 基础档案-项目 / 基础档案-经营单元 |
| 角色、权限、组织架构、公司主体 | 基础设置 | 角色权限 / 基础档案-组织架构 / 基础档案-公司主体 |
| 登录、界面、按钮、颜色、布局 | UI交互优化 | — |
| 性能、卡顿、慢、加载 | 技术 | 性能优化 |

> 以上是 AI 根据场景描述自动匹配的功能模块，请确认：
> 1. **准确** → 继续下一步
> 2. **需要修改** → 告诉我正确的一级/二级模块
> 3. **不确定** → 留空，由产品部后续补充

**修改规则：** CSM 可以修改一级或二级模块，最多修改 2 次。超过则留空。
